# Projeto-Construcao-de-MVP-Funcional-Sistema-de-Gestao-de-Tarefas
O DiaEmOrdem é um sistema web para gestão de tarefas pessoais (sistema To Do), desenvolvido com o objetivo de centralizar pendências diárias e reduzir a desorganização e o estresse causados por anotações dispersas.
